# Auditing.ai — AI Spend Audit Tool

A free tool that helps startups identify overspending on AI tools like Cursor, GitHub Copilot, ChatGPT, Claude, and more.



## GitHub
[github.com/PRATHMESHSURWASE/ai-spend-audit](https://github.com/PRATHMESHSURWASE/ai-spend-audit)

## Features
- Spend input form with localStorage persistence
- Audit engine with real pricing comparisons
- Per-tool savings breakdown
- AI-generated summary (Gemini Free API )
- Email capture and shareable link
- Free Credex consultation CTA if savings exceed $500/mo

## Tech Stack
- HTML, CSS, JavaScript (Vanilla)
- Tailwind CSS (CDN)
- Claude/Groq API for AI summary
- localStorage for state persistence
