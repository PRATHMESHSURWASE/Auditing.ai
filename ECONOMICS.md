# Unit Economics

## Business Model
Free tool as top-of-funnel lead generation for Credex consulting services.

## Revenue Model
- Free: AI spend audit tool (no charge)
- Paid: Credex consultation call ($200-500/session)
- Upsell: Ongoing SaaS spend management retainer ($500-2000/mo)

## Unit Economics (Estimated)
| Metric | Value |
|--------|-------|
| Cost to acquire one audit user | ~$2 (content/ads) |
| Audit to consultation conversion | ~5% |
| Average consultation value | $300 |
| LTV per audit user | $15 |
| Payback period | Immediate |

## Why This Works
The audit tool creates urgency — users see real dollar savings and are primed to pay for help implementing changes. The tool does the selling for you.

## Scale
At 1,000 audits/mo with 5% conversion = 50 consultations = $15,000/mo revenue from a free tool.
